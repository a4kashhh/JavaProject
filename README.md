# CCRM

Course and student management system in Java.

## Build & Run

```bash
chmod +x compile.sh
./compile.sh
java -cp out edu.ccrm.cli.Main
```

## Files

- `src/` - Java source code
- `sample-data/` - CSV test data  
- `compile.sh` - Build script

## Features

- Student management
- Course management  
- Enrollment tracking
- CSV import/export
- Data backup
